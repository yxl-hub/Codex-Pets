# 小鸡仔 Codex 桌宠

坤哥铁粉使用 Codex Vibe coding 了一个小鸡仔桌宠，让坤哥的卡通形象可以陪你工作。小鸡仔具备 9 个状态动作，自然也包括粉丝最爱的铁山靠。

这是一个基于 `小鸡仔.jpg` 制作的 Codex 桌面宠物开源包。仓库同时提供可直接安装的桌宠文件，以及可供复刻或二次开发的开发计划和提示词文档。

## 直接使用

将 `pets/xiao-ji-zai` 文件夹复制到你的 Codex 宠物目录：

```bash
mkdir -p "$HOME/.codex/pets"
cp -R pets/xiao-ji-zai "$HOME/.codex/pets/"
```

然后重启 Codex，在桌宠设置中选择 `小鸡仔`。

## 自行复刻或二次开发

核心文件：

- `PLAN.md`：Codex 桌宠生成、切帧、校验和打包流程。
- `Prompts.md`：角色约束、动作状态、逐行动画提示词和验收标准。
- `小鸡仔.jpg`：主参考素材图。
- `pets/xiao-ji-zai/pet.json`：Codex 桌宠配置。
- `pets/xiao-ji-zai/spritesheet.webp`：最终桌宠图集。

复刻时请优先遵守 `Prompts.md` 中的最高优先级：角色必须严格基于 `小鸡仔.jpg`，不能重新设计成泛化的小鸡、玩具或篮球角色。

## 桌宠状态

图集遵循 Codex 桌宠的 8 列 x 9 行结构，状态顺序为：

1. `idle`：单手转球，保持轻微待机动效。
2. `running-right`：背面朝前半蹲摆动，带铁山靠式节奏。
3. `running-left`：`running-right` 的镜像方向动作。
4. `waving`：胯下运球，头发随动作自然颤动。
5. `jumping`：带投篮感的起跳动作。
6. `failed`：篮球从手中掉落，角色低头注视篮球。
7. `waiting`：抱球等待，并带有眨眼动作。
8. `running`：正面单手运球，带跑动感。
9. `review`：盘腿坐着，认真观察手中的篮球。

## 开源前注意

如果你 fork 或二次发布，请确认你拥有参考图和生成图集的公开发布权。中间生成目录 `work/` 不建议提交，因为其中可能包含本机路径、生成记录和调试产物。
